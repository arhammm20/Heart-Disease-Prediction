❤️ Heart Disease Prediction System

Overview
This project is a Machine Learning-based Heart Disease Prediction System that predicts whether a person is at risk of heart disease based on medical parameters. It helps in early detection and decision support using data-driven insights.

Features
Predicts heart disease risk using trained ML model
User-friendly interface built with Streamlit
Handles real-time user input
Data preprocessing & feature scaling included
Accurate and fast predictions
🧠 Machine Learning Model
Model Used: Logistic Regression
Data Scaling: StandardScaler
Libraries: Pandas, NumPy, Scikit-learn

📊 Input Features
The model uses the following parameters:

Age
Sex
Chest Pain Type
Resting Blood Pressure
Cholesterol Level
Fasting Blood Sugar
Resting ECG
Maximum Heart Rate
Exercise-Induced Angina
Oldpeak
ST Slope

🛠️ Tech Stack
Python
Streamlit (Frontend UI)
Scikit-learn (ML Model)
Pandas & NumPy (Data Processing)
Joblib (Model Saving & Loading)
